* 웹표준이란?
 html태그를 사용해서 웹페이즈를 구축.
 -대소문자구분 안한다.
 -<p></p> 단점

 사용자 시스템의 환경과 무관하게
 보여주기 위해서는 웹표준이 필요하며,
 기존의 HTML 태그는 기술상으로 헛점이 많다.

 웹페이지를 누구나 가서 사용할 수 있게 하는것을
 웹접근성이라 하며,
 웹접근성을 용이하게 하는 가장 첫 번째 방법이
 웹표준이라 할수 있다.

 html -> xml

 기존의html문법을 흡수를 하고 xml문법도 사용할수 
 있게 xhtml이 나옴.

 html -> xhtml -> xml
 <strong></strong> -> 글자 진하게
 <진하게></진하게> -> xml에서는 명령어를 내가 직접 만들수 있다!

 웹표준위원회 W3C

 extensible html
 XHTML은 CSS와 많은 연관이 있다.

 CSS적용
 <div id=test>
 </div>

 XHTML문서를 만들기 위해서는
 1. 문서의 type을 선언한다. DTD 정의
 2. 스타일을 정의한다.
    <style type="text/css">
	#test{
	}
	.test{
	}
	div{
	}
    </style>
    스타일은 대소문자를 구분한다.
    주로 명령어는 소문자로 구성되어 있다.
    명령어 끝에 ;이 붙는다.
